# Что реализовано в этом проекте?

1. Шапка сайта (header):
   - При скролле header становится фиксированным.
   - При скролле и при нажатии на пункты меню, пункт меню выделяется активным классом.

2. Реализована функция плавного скролла (smooth scroll), при добавлении класса `yankor` на ссылку якорь.

3. Реализована функция скрытия текста.

4. Функция скрытия блоков для планшетов/телефонов.

5. Табы.

6. Фильтрация по категориям.

7. Попап.

8. Swiper slider.

9. Бургер-меню.

10. На телефонах/планшетах hover активен сразу, либо будет активен при нажатии на кнопку(для пк и тел).

11. Интегрирован Google Maps.

12. Адаптация (Media Queries) для телефонов/планшетов.

13. Masonry сетка.

14. Подгрузка изображений и карты при скроле для оптимизации сайта.

15. Скрипт, на определение поддерживает ли браузер более легких форматов изображения `avif`, `webp`.

16. Сео оптимизация, и теги для скрин ридеров, для людей с ограниченными возможностями.

17. Для многих кнопок использован формат изображения `svg`.

<br>

> В этом проекте, многие элементы страницы, сделаны так чтобы их можно было легко переиспользовать.

[Ссылка на сайт](https://eugklim.github.io/Bouncy/)

# Используемые технологии:

1. Flexbox.

2. Grid layout.
3. Применение переменных и миксинов (SCSS).
4. Семантическая верстка / мета-теги (SEO).
5. BEM методология.
6. Оптимизация изображений для уменьшения размера файлов и повышения скорости загрузки страниц:
   - Реализован скрипт на JS, который при заходе на сайт проверяет, если поддерживает `.avif`, тогда изображения (`.png`, `.jpg`) станут этого формата. Если не поддерживает `.avif`, то проверит, поддерживает ли браузер `.webp`, если и его не поддерживает, то вставит исходное, но оптимизированное за счет сжатия изображение.
7. Весь CSS и JS код минимизирован для лучшей производительности сайта. <br> Также для старых браузеров:
   - Для CSS добавлены вендорные префиксы.
   - Весь JS код преобразован плагином `Babel`.
   - Преобразование шрифтов в форматы `WOFF` и `WOFF2`.

# Установка и запуск проекта:

1. Клонировать репозиторий:

```ruby
git clone https://github.com/eugKlim/Bouncy.git
```

2. Установить зависимости:

```ruby
npm install
```

3. Запустить проект:

```ruby
npm run dev
```
